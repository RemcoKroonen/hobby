Dat hebben de makers van de server bekendgemaakt. Omdat ontwikkelaar Blizzard zelf geen server voor de oude versie van de game op BlizzCon heeft aangekondigd, voelt het team zich genoodzaakt zijn versie weer online te krijgen. De makers stellen hun code voor de server beschikbaar voor andere ontwikkelaars. Deze code zal door het team van de fanserver Elysium worden gebruikt, zodat deze server net als Nostalrius zal functioneren. De aangepaste versie van Elysium zal ook alle personages bevatten van spelers die op Nostalrius hebben gespeeld. Hierdoor kunnen spelers weer verder vanaf het punt waar ze gebleven waren.
